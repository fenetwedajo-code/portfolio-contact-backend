# Frontend By Toli

## hey dear 3-D this how you Should Push your Work

To contribute your work (for example: new feature, bugfix, or task completion):

1. **Clone the repository**  
   ```bash
   git clone https://github.com/3-D-engineers/frontendbytolera.git
   cd frontendbytolera
   ```

2. **Create a feature/task branch**  
   ```bash
   git checkout -b your-feature-branch
   ```

3. **Add your files and changes**

4. **Stage and commit your changes**  
   ```bash
   git add .
   git commit -m "Describe what you did"
   ```

5. **Pull latest changes to avoid conflicts**  
   ```bash
   git pull origin main
   # If you get a conflict, resolve it, then continue to add/commit.
   ```

6. **Push your work**  
   ```bash
   git push origin your-feature-branch
   # Or push directly to main if allowed:
   # git push origin main
   ```

7. **Open a Pull Request on GitHub**  
   - Go to the repo page.
   - Click "Compare & pull request" for your branch.
   - Submit your pull request for review.

8. **Wait for review and merge (done by a maintainer or with your permission).**

---

## Notes
- Always pull (`git pull origin main`) before pushing if others may have updated the repo.
- Use clear commit messages.
- Ask for help if you run into any conflict or error.
